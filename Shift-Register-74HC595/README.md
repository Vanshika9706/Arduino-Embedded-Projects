# 74HC595 Shift Register Experiments with Arduino

## Overview

This project demonstrates multiple experiments using the 74HC595 shift register to expand digital outputs of an Arduino.
The shift register allows control of multiple LEDs using only three Arduino pins.

## Components

* Arduino Uno
* 74HC595 Shift Register IC
* LEDs
* Resistors
* Breadboard
* Jumper wires

## Experiments Implemented

### Binary Counter

Displays a binary counting sequence using LEDs connected through the shift register.

### Binary Counter with 7-Segment Display

Uses the shift register to control a 7-segment display and show numeric values.

### Forced Reset

Demonstrates resetting the output state of the shift register.

### Forced Reset Variant

Alternate implementation of reset logic.

### Knight Rider Pattern

Creates a scanning LED pattern similar to the Knight Rider light effect.

## Learning Outcomes

* Understanding serial-to-parallel conversion
* Interfacing external ICs with Arduino
* Expanding microcontroller output pins
* Controlling LED patterns using shift registers
